# WackyBanana Setup Instructions

## Step 1: Clone the Repository
```bash
git clone github.com/mismaeelqazi/WackyBanana.git
```

## Step 2: Install Dependencies
```bash
pip install -r requirements.txt
```

## Step 3: Run the Script
```bash
python banana.py
```
